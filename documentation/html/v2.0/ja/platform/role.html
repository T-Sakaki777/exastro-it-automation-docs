2.3.3. ロール紐づけ¶

{{ #2 記載お願いいたします。 }}
ロール管理では下記２つの使用方法を行うことが可能です。
  • ロール追加・ワークスペース紐づけ
    ロールごとにワークスペースへのアクセス権限をメンテナンスできます。
  • ユーザー・ロール紐づけ
    ユーザ毎にアクセスを許可するロールを付与することで、ユーザごとにワークスペースへのアクセスを制御することができます。

ロール追加・ワークスペース紐づけ¶

  1. Exastro Platformにオーガナイゼーション管理者 または _${ws-id}-adminロールに紐づくユーザーでログインします。
  2. Exastro Platformのメニューより ロール管理 をクリックします。

    注釈

    オーガナイゼーション管理者以外でログインしている時は、メニューに ロール管理 は表示されません。
  3. ロール一覧画面が表示されるので、作成 ボタンをクリックします。
  4. 新規ロール画面が表示されるので、ロールの情報を入力し、登録 ボタンをクリックします。

    注釈

    「使用ワークスペース」にワークスペースの一覧が表示されるので、当該ロールに紐づけるワークスペースを選択します。

ユーザー・ロール紐づけ¶

  1. Exastro Platformにオーガナイゼーション管理者 または _${ws-id}-adminロールに紐づくユーザーでログインします。
  2. Exastro Platformのメニューより ロール管理 をクリックします。

    注釈

    オーガナイゼーション管理者、_${ws-id}-adminロールに紐づくユーザー以外でログインしている時は、メニューに ロール管理 は表示されません。
  3. ロール一覧画面が表示されるので、ユーザーとロールの紐づけを変更したいロールを選択し ユーザー ボタンをクリックします。
  4. ロール付与・解除画面が表示されるので、ロールとの紐づけを変更したいユーザーを選択し、 適用 ボタンをクリックします。
