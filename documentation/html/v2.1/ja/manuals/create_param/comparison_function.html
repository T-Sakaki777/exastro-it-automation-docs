比較機能¶

本章では比較機能について説明します。

比較機能について¶

比較機能とは、ITAのメニュー作成機能で作成したパラメータシートを比較し、差分を出力する機能です。

本機能は、「パラメータシート（ホスト/オペレーションあり）」を対象としています。
パラメータシートの詳細については、「利用手順マニュアル メニュー作成機能」を参照してください。
アクセス権/アクセス許可ロールについては、「利用手順マニュアル データレコード毎の
ロールベースアクセス制御」を参照してください。
